# Music Generator App

A Next.js application that generates a fake music catalog with songs, artists, albums, reviews, likes, covers and audio previews.

## Features

- Song generation based on region, seed and likes
- Table and gallery views
- Infinite scroll gallery
- Song details
- Generated SVG covers
- Audio preview playback
- React Query for data fetching
- Material UI components

## Tech Stack

- Next.js
- React
- Material UI
- TanStack React Query
- Faker.js
- Tone.js
- Zod
